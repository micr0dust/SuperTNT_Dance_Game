# SuperTNT dance game
這遊戲原本是以前在用VB製作專題時寫的遊戲，現在用網頁復刻，也改良了一些寫法和bug，並支援電腦和部分手機版。(建議用Chrome開啟)

### 規則:
在時間歸零時站在指定色塊上以存活，並可放置TNT陷害對方。

### 電腦版控制:
1P:WASD+G
2P:上下左右+[/]
demo: https://wuilliam104286.github.io/SuperTNT_Dance_Game/
